# Chart JS

The ChartJSInput component is a modern, interactive charting solution built using React and powered by the highly flexible Chart.js library.
It provides seamless support for multiple chart types including Bar (Horizontal & Vertical), Line, Area, and Pie charts, making it ideal for building analytical dashboards and data visualization interfaces inside Mendix applications.

This component is optimized for Mendix custom widget integration and includes advanced visualization capabilities such as responsive rendering, zooming, exporting, dataset customization, legends, tooltips, and dynamic styling.

**Features**

**General Chart Features**
Supports multiple chart types:
      
      •	Bar Chart (Vertical & Horizontal)  

      •	Line Chart 

      •	Area Chart 

      •	Pie Chart 

• Responsive and adaptive chart rendering for desktop and mobile layouts.

• Smooth animations and interactive hover effects.

• Dynamic dataset binding from Mendix entities or data source attributes.

• Fully customizable chart titles, legends, axes, and labels.

• Supports multiple datasets within a single chart.

• Automatic color generation for datasets and chart segments.

## Bar Chart Features

• Supports both vertical and horizontal orientations

• Configurable stacked and grouped bar layouts

• Adjustable bar thickness, border radius, and spacing

• Axis label customization and grid visibility controls

• Interactive tooltip support for precise data inspection

## Line & Area Chart Features

• Smooth curved or straight line rendering

• Area fill support with opacity customization

• Multiple line datasets with independent styling

• Point styling with hover radius and interaction effects

• Configurable tension and interpolation settings

## Pie Chart Features

• Dynamic slice generation based on dataset values

• Customizable slice colors and borders

• Interactive legends and tooltip support

• Percentage-based visual representation

• Supports doughnut-style rendering customization

## Interactivity & User Experience

• Zoom In and Zoom Out functionality for detailed analysis

• Fullscreen mode support for large-scale visualization

• Download chart as image for reporting and sharing

• Real-time chart updates when datasource changes

• Interactive legends with dataset toggle visibility

• Tooltip customization for enhanced readability
